# Installation

+ Download packages:
```bash
pacman -S fish kitty niri starship waybar helix rofi awww 
```

+ Set up shell:
```bash
chsh -s /usr/bin/fish <user>
```

+ Copying configuration files:
```bash
cd
git clone https://github.com/Perelmeshcka/niri-dots.git
cp -r niri-dots/dot-config/* .config/
```
